# 🕷️ Silk Lounge

**Silk Lounge** is a 2D platformer set inside a stylish 1930s speakeasy world, where a spider-themed protagonist moves through a moody lounge filled with gang tension, cops, and web-slinging movement. The game blends platforming, themed level design, enemy encounters, and atmospheric visuals into a short but polished GameMaker experience.

https://collectorsincorporated.itch.io/silk-lounge

---

## 🎮 About the Game

In **Silk Lounge**, the player explores a vintage underground lounge inspired by 1930s crime and jazz-era aesthetics. The world is built around two rival factions: **The Charleston Web** and the **Yellowjack Hive**. The player must navigate through the environment, avoid danger, and use movement mechanics like jumping and web-slinging to progress through the level.

The project was developed as part of a game development course using **GameMaker Studio**.

## 🕹️ Controls

| Action | Key |
|---|---|
| Move Left | `A` / `Left Arrow` |
| Move Right | `D` / `Right Arrow` |
| Jump | `Space` |
| Web Sling | Assigned in-game input |
| Restart Room | `Tab + R` |
| Next Room | `Tab + N` |
| Add Coins | `Tab + C` |
| End Game | `Tab + Esc` |
